<h2> 🎨 Modern Art Prints E-Commerce Platform </h2>

A full-stack ASP.NET MVC e-commerce web application for selling modern art prints online.
The platform provides a complete shopping experience for users along with a powerful admin dashboard for managing products, orders, users, and website content.

📌 **Project Overview**

Modern Art Prints is an online store designed to showcase and sell high-quality modern art prints through a clean and responsive interface.

The application includes:

- User authentication and authorization
- Product browsing and search
- Shopping cart and wishlist
- Secure checkout using an external payment API
- Order tracking and invoice generation
- Product reviews and testimonials
- Full admin dashboard with analytics and moderation tools
